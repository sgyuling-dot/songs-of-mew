# Songs of Mew 🐱

一款类丝之歌风格的平台跳跃银河城小游戏。

## 运行方式

直接用浏览器打开 `index.html` 即可（无需安装任何依赖）。

## 操作说明

| 操作 | 按键 |
|------|------|
| 左右移动 | `← →` 或 `A D` |
| 跳跃（支持二段跳） | `空格` / `W` / `↑` |
| 近战攻击（挥爪） | `J` 或 `Z` |
| 重新开始 | `R` |

## 游戏内容

- 控制猫猫角色穿越平台，躲避/击杀敌人，到达终点传送门通关
- 共 6 个巡逻敌人，3 滴血，支持二段跳
- 矢量几何画风，深色丝之歌风格配色

## 技术栈

- HTML5 Canvas + 纯 JavaScript（无框架）
- 矢量绘图（无图片素材）
